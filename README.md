# sentiment
This project performs sentiment analysis on a Twitter dataset using a local NLP approach with VADER (Valence Aware Dictionary for Sentiment Reasoning).
It is designed as a lightweight alternative to cloud-based sentiment services such as Azure Text Analytics.

The project includes:
Data ingestion
Sentiment scoring
Data export
Visualization
Power BI dashboard
Final insights
This project fulfills the requirements of a sentiment analysis assignment, including architecture, code, data processing, and visualization.
